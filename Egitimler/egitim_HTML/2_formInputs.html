Form Giriş Alanları

HTML'de formlar oluşturmak için çeşitli giriş alanları kullanılabilir. Bu giriş alanları, kullanıcılardan metin, e-posta, dosyalar ve diğer bilgileri toplamanıza olanak tanır.

Giriş Alanı Türleri:


  • Metin Girişi: input type="text"
  • E-posta Girişi: input type="email"
  • Şifre Girişi: input type="password"
  • Seçenek Kutuları: input type="checkbox"
  • Radyo Düğmeleri: input type="radio"
  • Çok Satırlı Metin Girişi: textarea
  • Dosya Seçimi: input type="file"
  • Gönderme Butonu: input type="submit"
  • Reset Butonu: input type="reset"

Giriş Alanı Öznitelikleri:


  • id: Giriş alanına benzersiz bir kimlik verir.
  • name: Giriş alanının adını belirtir. Bu isim, form verilerini sunucuya göndermek için kullanılır.
  • required: Bu alanın doldurulmasını zorunlu hale getirir.
  • placeholder: Alana varsayılan metin gösterir.
  • value: Alanın varsayılan değerini belirtir.
  • rows: Çok satırlı metin alanları için satır sayısını belirtir.
  • cols: Çok satırlı metin alanları için sütun sayısını belirtir.
  • accept: Dosya seçim alanları için kabul edilebilecek dosya türlerini belirtir.

Seçenek Kutuları ve Radyo Düğmeleri:


Seçenek kutuları ve radyo düğmeleri, kullanıcılardan birden fazla seçenekten birini seçmelerini sağlar.

Gruplar:


Seçenek kutularını ve radyo düğmelerini gruplayarak kullanıcının birden fazla seçenek seçmesine izin verebilirsiniz. Bunu yapmak için name özniteliğine aynı değeri verirsiniz.
