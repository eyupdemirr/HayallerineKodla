CSS ile Entegrasyon

HTML, web sayfalarının temelini oluştururken, CSS bu sayfalara stil ve işlevsellik kazandırmak için kullanılır. İkisini birlikte kullanarak, etkileşimli, görsel açıdan çekici ve kullanımı kolay web siteleri oluşturabilirsiniz.

CSS Entegrasyon Yöntemleri:

1. Harici CSS Dosyası:


  • En yaygın yöntemdir.
  • Tüm sayfalar için tek bir CSS dosyası oluşturabilir ve link etiketi ile HTML'e bağlayabilirsiniz.

2. Dahili CSS:


  • style etiketi ile HTML sayfasının head bölümüne CSS kodları ekleyebilirsiniz.
  • Küçük projeler veya sayfaya özel stiller için uygundur.

3. Sıralı CSS:


  • style etiketi ile HTML öğelerine doğrudan CSS kodları ekleyebilirsiniz.
  • Nadiren kullanılır ve sadece belirli öğeler için stil vermek için uygundur.

CSS Seçicileri:


CSS'de, belirli HTML öğelerini hedeflemek ve onlara stil uygulamak için seçiciler kullanılır. En yaygın seçici türleri şunlardır:

  • Element Seçicisi: h1, p, img gibi HTML öğelerini hedefler.
  • ID Seçicisi: #benzersiz-kimlik gibi HTML öğelerinin kimliklerini kullanır.
  • Sınıf Seçicisi: .sinif-adi gibi HTML öğelerinin sınıflarını kullanır.
  • Çocuk Seçicisi: a > p gibi bir öğenin altındaki öğeleri hedefler.
  • Torun Seçicisi: ul li a gibi bir öğenin altındaki alt öğeleri hedefler.
  • Özellik Seçicisi: a[href="https://www.example.com"] gibi belirli bir özelliğe sahip öğeleri hedefler.
