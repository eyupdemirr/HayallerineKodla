DOM (Document Object Model):

Web sayfaları, HTML kodları ile oluşturulur. DOM, bu HTML kodlarını bir belge olarak temsil eden ve bu belge üzerinde programatik olarak değişiklik yapmamızı sağlayan bir arayüzdür. Tıpkı bir evin farklı odalarına ve eşyalarına erişmek için bir harita kullandığımız gibi, DOM da bir web sayfasının farklı öğelerine ve özelliklerine erişmek için bir harita görevi görür. 


        // Sayfadaki bir başlığı seçip rengini kırmızıya değiştirelim
        var baslik = document.getElementById("baslik"); // ID ile öğeyi seçelim
        baslik.style.color = "red"; // Rengini kırmızıya ayarlayalım

DOM Seçicileri:


DOM'da farklı öğelere ulaşmak için çeşitli seçiciler kullanabiliriz.


  • getElementById: ID ile öğeyi seçer.
  • getElementsByTagName: Belirli bir etiket adıyla tüm öğeleri seçer.
  • getElementsByClassName: Belirli bir sınıf adıyla tüm öğeleri seçer.
  • querySelector: CSS seçici kullanarak tek bir öğeyi seçer.
  • querySelectorAll: CSS seçici kullanarak birden fazla öğeyi seçer.

        // Sayfadaki tüm paragrafları seçip arka plan rengini griye değiştirelim
        var paragraflar = document.getElementsByTagName("p"); // Tüm paragrafları seçelim

        for (var i = 0; i < paragraflar.length; i++) {
          paragraflar[i].style.backgroundColor = "gray"; // Arka plan rengini griye ayarlayalım
        }

DOM Özellikleri:


DOM öğelerinin çeşitli özellikleri vardır. Bu özellikleri kullanarak öğelerin görünümünü, içeriğini ve davranışını değiştirebiliriz. 


        // Sayfadaki bir bağlantının href özelliğini ve metnini değiştirelim
        var baglanti = document.getElementById("baglanti"); // ID ile öğeyi seçelim
        baglanti.href = "https://www.org.com"; // href özelliğini değiştirelim
        baglanti.innerText = "Yeni Bağlantı Metni"; // Metnini değiştirelim

DOM Olay İşleyicileri:


DOM öğeleri, tıklama, fare hareketleri, klavye basma gibi olaylara tepki verebilir. Bu olaylara tepki vermek için olay işleyicileri kullanabiliriz. 


        // Sayfadaki bir butona tıklandığında bir mesaj yazdıralım
        var buton = document.getElementById("buton"); // ID ile öğeyi seçelim
        buton.addEventListener("click", function() {
          alert("Butona tıklandı!");
        });

JavaScript ile Neler Yapılabilir?


  • Kullanıcı etkileşimleri: Kullanıcıların tıklamalarına, fare hareketlerine ve form girişlerine yanıt verebilirsiniz.
  • Animasyonlar ve görsel efektler: Sayfalarınızda dinamik animasyonlar ve görsel efektler oluşturabilirsiniz.
  • Dinamik içerik: Kullanıcı girdisine veya sunucu verisine göre değişen içerikler oluşturabilirsiniz.
  • Oyunlar ve interaktif uygulamalar: Eğlenceli ve etkileşimli oyunlar ve uygulamalar geliştirebilirsiniz.
  • Web API'leri ile entegrasyon: Haritalar, videolar ve sosyal medya gibi web API'lerini kullanarak web sitenize yeni fonksiyonlar ekleyebilirsiniz.
