Operatörler

Operatörler, değişkenler ve değerler üzerinde çeşitli işlemler yapmamızı sağlayan sembollerdir. Matematiksel işlemler, karşılaştırma işlemleri, mantıksal işlemler gibi birçok farklı türde operatör vardır.

Matematiksel Operatörler


  • +: Toplama
  • -: Çıkarma
  • *: Çarpma
  • /: Bölme
  • %: Kalan bulma
  • ++: Artırma
  • --: Azaltma

        var x = 10;
        var y = 5;
                                
        var toplam = x + y;
        var cikarma = x - y;
        var carpim = x * y;
        var bolum = x / y;
        var kalan = x % y;
                                
        console.log("Toplam: " + toplam); // Çıktı: Toplam: 15
        console.log("Çıkarma: " + cikarma); // Çıktı: Çıkarma: 5
        console.log("Çarpım: " + carpim); // Çıktı: Çarpım: 50
        console.log("Bölüm: " + bolum); // Çıktı: Bölüm: 2
        console.log("Kalan: " + kalan); // Çıktı: Kalan: 0
                                
        x++; // x'i 1 artırır
        y--; // y'yi 1 azaltır
                                
        console.log("Yeni x: " + x); // Çıktı: Yeni x: 11
        console.log("Yeni y: " + y); // Çıktı: Yeni y: 4

Operatör Önceliği:


Bazı operatörler diğerlerinden daha önce işlenir. Operatör önceliğini bilmek, kodumuzun doğru şekilde çalışmasını sağlamak için önemlidir. 


        var sonuc = 10 + 5 * 2;
        console.log("Sonuç: " + sonuc); // Çıktı: Sonuç: 20

Karşılaştırma Operatörler


  • ==: Eşitlik
  • !=: Eşitsizlik
  • <: Küçüktür
  • <=: Küçüktür veya eşittir
  • >: Büyüktür
  • >=: Büyüktür veya eşittir

        var x = 10;
        var y = 5;
        
        var esitlik = x == y;
        var esitsizlik = x != y;
        var kucuk = x < y;
        var kucukEsit = x <= y;
        var buyuk = x > y;
        var buyukEsit = x >= y;
        
        console.log("Eşitlik: " + esitlik); // Çıktı: Eşitfalse
        console.log("Eşitsizlik: " + esitsizlik); // ÇıEşitsizlik: true
        console.log("Küçüktür: " + kucuk); // Çıktı: Küçükfalse
        console.log("Küçüktür veya Eşittir: " + kucukEsit)Çıktı: Küçüktür veya Eşittir: false
        console.log("Büyüktür: " + buyuk); // Çıktı: Büyüktrue
        console.log("Büyüktür veya Eşittir: " + buyukEsit)Çıktı: Büyüktür veya Eşittir: true

Mantıksal Operatörler


  • &&: Ve (AND)
  • ||: Veya (OR)
  • !: Değil (NOT)

        var x = 10;
        var y = 5;
        var z = 20;
        
        var veIslemi = x < y && y < z;
        var veyaIslemi = x > y || y > z;
        var degilIslemi = !veIslemi;
        
        console.log("Ve İşlemi: " + veIslemi); // Çıktıİşlemi: false
        console.log("Veya İşlemi: " + veyaIslemi); // ÇıVeya İşlemi: true
        console.log("Değil İşlemi: " + degilIslemi); // ÇıDeğil İşlemi: true
