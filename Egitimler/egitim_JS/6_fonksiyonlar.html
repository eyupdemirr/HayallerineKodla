Fonksiyonlar

Hayal edin, bir kutunuz var ve bu kutuya "Merhaba Dünya!" yazdırmak için bir talimat vermek istiyorsunuz. Bu talimatı kutuya kaydedip, her zaman aynı sonucu alacak şekilde kullanabilirsiniz. İşte fonksiyonlar tam olarak bu işe yarar! 


        // Merhaba Dünya! mesajını yazdıran bir fonksiyon oluşturalım
        function merhabaDunya() {
          console.log("Merhaba Dünya!");
        }

        // Fonksiyonumuzu çağıralım 
        merhabaDunya(); 
        
        // Çıktı: Merhaba Dünya!

Fonksiyon Parametreleri:


Fonksiyonlar, farklı değerler alarak çalışabilir. Bu değerleri fonksiyon parametreleri olarak tanımlayabiliriz. 


        // Bir sayıya toplama işlemi yapan bir fonksiyon oluşturalım
        function toplama(sayi1, sayi2) {
          var toplam = sayi1 + sayi2;
          return toplam;
        }

        // Fonksiyonumuzu çağıralım ve sonucu değişkene atayalım
        var sonuc = toplama(5, 3);
        console.log("Sonuç: " + sonuc); // Çıktı: Sonuç: 8

Fonksiyon Dönüş Değerleri:


Fonksiyonlar, işlem sonucunda bir değer döndürebilir. Bu değeri return anahtar kelimesi ile belirtiriz. 


        // Bir metni büyük harfe çeviren bir fonksoluşturalım
        function buyukHarfeCevir(metin) {
          return metin.toUpperCase();
        }
        
        // Fonksiyonumuzu çağıralım ve sonucu değişkene atayalım
        var buyukMetin = buyukHarfeCevir("Merhaba");
        console.log("Büyük Metin: " + buyukMetin); // ÇıBüyük Metin MERHABA

Fonksiyon Türleri:


Fonksiyonlar, tanımlanma şekillerine göre farklı türlere ayrılabilir.

  • Bildirimli Fonksiyonlar: function anahtar kelimesi ile bildirilir.
  • İfade Fonksiyonları: => operatörü ile kısaltılmış şekilde bildirilir.

        // Bildirimli fonksiyon
        function carpma(x, y) {
          return x * y;
        }

        // İfade fonksiyonu
        var carpma = (x, y) => x * y;

        // Fonksiyonları çağıralım
        var sonuc1 = carpma(10, 2);
        var sonuc2 = carpma(5, 4);

        console.log("Sonuç 1: " + sonuc1); // Çıktı: Sonuç 1: 20
        console.log("Sonuç 2: " + sonuc2); // Çıktı: Sonuç 2: 20

Fonksiyon Kapsamı:


Fonksiyonlar kendi kapsamlarına sahiptir. Fonksiyon içinde tanımlanan değişkenler, sadece o fonksiyonda erişilebilir. 


        // Fonksiyon içinde tanımlanan değişkene erişilemez
        var globalDegisken = 10;
        
        function yerelDegisken() {
          var yerelDegisken = 20;
          console.log(globalDegisken); // 10'a erişebilir
          console.log(yerelDegisken); // 20'ye erişebilir
        }
        
        yerelDegisken();
        console.log(yerelDegisken); // Hata: yerelDegideğişkenine erişilemez
