Değişkenler ve Veri Türleri

Hayal edin, bir kutunuz var ve bu kutuya "Merhaba Dünya!" yazmak istiyorsunuz. Bu kutuyu isimlendirmek için "mesaj" adını verebilirsiniz. Bu şekilde, kutunuzun içindeki metne her zaman "mesaj" yazarak erişebilirsiniz. İşte değişkenler tam olarak bu işe yarar! 


        // Mesaj kutumuzu oluşturalım
        var mesaj = "Merhaba Dünya!";

        // Kutumuzu açıp içindeki metni görelim
        console.log(mesaj); // Çıktı: Merhaba Dünya!

Değişkenleri Değiştirme:


Değişkenlerimize atadığımız değerleri daha sonra değiştirebiliriz. 


        var mesaj = "Merhaba";
        mesaj = "Hoşgeldiniz";
        console.log(mesaj); // Çıktı: Hoşgeldiniz

Sabit Değerler (const):


Bazı değerlerin değişmesini istemeyebiliriz. Bu durumlarda const anahtar kelimesini kullanarak sabit değerler oluşturabiliriz. Sabit değerler atandıktan sonra değiştirilemez. 


        const PI = 3.14159;
        // PI değerini değiştiremeyiz
        PI = 2.71828; // Hata: PI sabit bir değerdir ve değiştirilemez

Değişken Kapsamı (Scope):


Değişkenlerimizin erişilebilirlik alanları vardır. Bir değişkenin kapsamı, tanımlandığı yer ile sınırlıdır. 


        // Blok içinde tanımlanan değişken
        var mesaj = "Merhaba";
        if (true) {
          var isim = "Ali"; // Sadece bu blok içerişilebilir
        }
        console.log(isim); // Hata: isim değişkenine erişilemez

Değişken Bildirimi (Declaration):


Değişkenlerimizi var, let veya const anahtar kelimeleri ile bildirebiliriz. 


        // var ile değişken bildirimi
        var mesaj = "Merhaba";
        
        // let ile değişken bildirimi
        let isim = "Ali";
        
        // const ile sabit değer bildirimi
        const PI = 3.14159;

Veri Türleri

Değişkenlerimiz farklı türde değerler tutabilir. Tıpkı bir kutunun farklı nesneler tutabileceği gibi, değişkenlerimiz de metin, sayı, mantık değeri gibi farklı türde veriler tutabilir.

Metin (String)


Metin türü, harfler, kelimeler ve semboller içeren değerleri saklar.


        var adi = "Ali";
        var soyadi = "Veli";

        console.log(adi + " " + soyadi); // Çıktı: Ali Veli

Sayı (Number):


Sayı türü, tam sayılar ve ondalıklı sayılar gibi sayısal değerleri saklar.


        var yas = 25;
        var boy = 1.75;
        
        console.log("Yaşım: " + yas + ", Boyum: " + boy)Çıktı: Yaşım: 25, Boyum: 1.75

Mantık (Boolean):


Mantık türü, doğru(true) veya yanlış(false) değerleri saklar.


        var canli = true;
        var aktif = false;
        
        console.log("Canlı mıyım?: " + canli + ", Aktif miy" + aktif); // Çıktı: Canlı mıyım?: true, Aktif miyfalse

Veri Türlerini Kontrol Etme:


Hangi türde bir veriye sahip olduğumuzu kontrol etmek için typeof operatörünü kullanabiliriz. 


        var mesaj = "Merhaba";
        var yas = 25;
        var canli = true;
        
        console.log(typeof mesaj); // Çıktı: string
        console.log(typeof yas); // Çıktı: number
        console.log(typeof canli); // Çıktı: boolean
