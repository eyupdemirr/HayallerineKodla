Koşul İfadeleri

Koşul ifadeleri, programınızın belirli koşullara bağlı olarak farklı yollar izlemesini sağlayan temel programlama yapı taşlarından biridir. C# dilinde en yaygın kullanılan koşul ifadeleri şunlardır:


  • if: Belirli bir koşulun doğru olup olmadığını kontrol eder ve koşul doğruysa belirli kodları yürütür.
  • else: if koşulunun yanlış olması durumunda yürütülmesi gereken kodları belirtir.
  • else if: Birden fazla koşulu sırayla kontrol etmenizi sağlar.
  • switch: Birden fazla olası senaryoyu değerlendirmek için kullanılır.

if Koşulu

Bu kod, yas değişkeninin değerine göre "Yetişkinsiniz!" veya "Yetişkin değilsiniz!" mesajını ekrana yazar.


        int yas = 18;
        if (yas >= 18)
        {
          Console.WriteLine("Yetişkinsiniz!");
        }
        else
        {
          Console.WriteLine("Yetişkin değilsiniz!");
        }

else if Koşulu

Bu kod, not değişkeninin değerine göre "Mükemmel!", "Çok İyi!", "İyi!" veya "Başarısız!" mesajını ekrana yazar.


        int not = 85;
        if (not >= 90)
        {
          Console.WriteLine("Mükemmel!");
        }
        else if (not >= 80)
        {
          Console.WriteLine("Çok İyi!");
        }
        else if (not >= 70)
        {
          Console.WriteLine("İyi!");
        }
        else
        {
          Console.WriteLine("Başarısız!");
        }

else Koşulu

Koşul yapılarında istenilen koşullara uymayan durumlarda çalışması istenilen kodların yazıldığı alanı ifade eder. Değilse anlamına gelir ve her koşul yapısında kullanılması zorunlu değildir.

Switch Koşulu

Bu kod, meyve değişkeninin değerine göre "Elma seçtiniz!", "Armut seçtiniz!", "Muz seçtiniz!" veya "Bilinmeyen meyve!" mesajını ekrana yazar. 


        string meyve = "elma";
        switch (meyve)
        {
          case "elma":
            Console.WriteLine("Elma seçtiniz!");
            break;
          case "armut":
            Console.WriteLine("Armut seçtiniz!");
            break;
          case "muz":
            Console.WriteLine("Muz seçtiniz!");
            break;
          default:
            Console.WriteLine("Bilinmeyen meyve!");
            break;
        }

Veri Tipleri

Veri tipleri, programlamada farklı türde verileri temsil etmek için kullanılan yapılardır. Sayısal, metinsel, mantıksal, dizi, nesne gibi çeşitli veri tipleri mevcuttur.

Sayısal Veri Tipleri

Sayısal veri tipleri, tam sayılar, ondalıklı sayılar ve karmaşık sayılar gibi sayısal değerleri saklamak için kullanılır.

Metinsel Veri Tipleri

Metinsel veri tipleri, harfler, kelimeler ve cümleler gibi metin tabanlı verileri saklamak için kullanılır.

Mantıksal Veri Tipleri

Mantıksal veri tipleri, doğru veya yanlış gibi mantıksal değerleri saklamak için kullanılır.
