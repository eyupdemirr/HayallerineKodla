Döngüler

Döngüler, programımızda belirli bir kod bloğunu tekrar tekrar çalıştırmamızı sağlar. Tekrarlayan işlemleri otomatikleştirmek ve kodumuzu daha kısa ve verimli hale getirmek için döngülerden faydalanabiliriz.

for Döngüsü


for döngüsü, belirli bir sayıda tekrarlama yapmak için kullanılır. 


        // 1'den 10'a kadar sayıları yazdıralım
        for (var i = 1; i <= 10; i++) {
          console.log(i);
        }

while Döngüsü


while döngüsü, bir koşul doğru olduğu sürece tekrarlama yapar. 


        // Bir sayı 100'e ulaşana kadar rastgele sayılar üretelim
        var sayi = 0;

        while (sayi < 100) {
          sayi = Math.floor(Math.random() * 101); // 0 ile 100 arasında rastgele bir sayı üretelim
          console.log("Rastgele Sayı: " + sayi);
        }

do-while Döngüsü


do-while döngüsü, bir koşul doğru olduğunda tekrarlama yapar, ancak koşul yanlış olsa bile döngü en az bir kez çalıştırılır. 


        // Bir şifre girilene kadar kullanıcıdan şifre isteyelim
        var sifreGirilen = false;
        var sifre;

        do {
          sifre = prompt("Şifrenizi giriniz:");
        
          if (sifre === "1234") {
            sifreGirilen = true;
            console.log("Doğru şifreyi girdiniz!");
          } else {
            console.log("Hatalı şifre!");
          }
        } while (!sifreGirilen);

        //Bu kod, kullanıcıdan "1234" şifresini girmesini isteyecek ve şifre doğru girilene kadar bu işlemi tekrarlayacaktır.

for...in Döngüsü


for...in döngüsü, bir nesnenin tüm özelliklerini sırayla dolaşmak için kullanılır. 


        // Bir öğrencinin tüm bilgilerini yazdıralım
        var ogrenci = {
          isim: "Ali",
          soyisim: "Veli",
          yas: 18,
          okul: "Atatürk İlkokulu"
        };
        
        for (var anahtar in ogrenci) {
          console.log(anahtar + ": " + ogrenci[anahtar]);
        }                            
                        

        //Output
        isim: Ali
        soyisim: Veli
        yas: 18
        okul: Atatürk İlkokulu

for...of Döngüsü


for...of döngüsü, bir dizinin tüm öğelerini sırayla dolaşmak için kullanılır. 


        // Bir meyve dizisinin tüm öğelerini yazdıralım
        var meyveler = ["Elma", "Muz", "Portakal"];
        
        for (var meyve of meyveler) {
          console.log(meyve);
        }                                                
                        

        //Output
        Elma
        Muz
        Portakal
