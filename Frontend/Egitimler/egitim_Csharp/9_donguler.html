Döngüler

Döngüler, C# programlama dilinde bir işlemi birden fazla kez tekrarlamak için kullanılan temel programlama yapı taşlarından biridir.

Döngü Türleri:

  • For: Belirli bir sayıda tekrar yapılacak işlemleri kontrol etmek için kullanılır.
  • While: Bir koşul doğru olduğu sürece işlemleri tekrarlamak için kullanılır.
  • Do-while: Bir koşul yanlış olduğu sürece işlemleri tekrarlamak için kullanılır.
  • Foreach: Dizilerdeki veya koleksiyonlardaki öğeleri sırayla işlemeyi sağlar.

For Döngüsü


        for (int i = 0; i < 5; i++)
        {
          Console.WriteLine(i);
        }

Bu kod, 0'dan 4'e kadar olan sayıları sırayla ekrana yazar.

While Döngüsü:


        int x = 1;
        while (x <= 10)
        {
          Console.WriteLine(x);
          x++;
        }

Bu kod, x değişkeninin değeri 10'dan küçük veya eşit olduğu sürece x değerini ekrana yazar ve ardından x değerini 1 artırır.

Do-while Döngüsü:


        int y = 1;
        do
        {
          Console.WriteLine(y);
          y++;
        }
        while (y <= 5);

Bu kod, y değişkeninin değeri 5'ten büyük olana kadar y değerini ekrana yazar ve ardından y değerini 1 artırır.

Foreach Döngüsü:


        string[] isimler = {"Ahmet", "Mehmet", "Ayşe"};
        foreach (string isim in isimler)
        {
          Console.WriteLine(isim);
        }

Bu kod, isimler dizisindeki her bir elemanı sırayla ekrana yazar.
