MATH() Fonksiyonu

Matematiksel işlemleri gerçekleştirmek için yerleşik bir sınıf olan Math sınıfı bulunur. Çok sayıda matematiksel işlemi kolayca yapabilmek için çok sayıda yöntem mevcuttur.

Çeşitli Matematiksel Yöntemler

  • Yuvarlama: Floor(), Ceiling(), Round(), Abs()
  • Güç alma: Pow()
  • Logaritma: Log(), Log10(), Log2()
  • Trigonometrik fonksiyonlar: Sin(), Cos(), Tan(), Asin(), Acos(), Atan()
  • Hiperbolik fonksiyonlar: Sinh(), Cosh(), Tanh(), Asinh(), Acosh(), Atanh()
  • Mutlak değer: Abs()
  • Karekök: Sqrt()
  • Kübik kökü: Cbrt()
  • Üstün logaritma: Exp()
  • Faktöriyel: Factorial()
  • Gamma fonksiyonu: Gamma()
  • Beta fonksiyonu: Beta()
  • Eğrisel integral: Quad()
  • Diferansiyel denklemler: Odeint()

Kod Örnekleri

Yuvarlama:


        double sayi = 5.4321;

        double yukari_yuvarlama = Math.Ceiling(sayi); // 6
        double asagi_yuvarlama = Math.Floor(sayi); // 5
        double yakin_yuvarlama = Math.Round(sayi); // 5

        Console.WriteLine("Yukarı yuvarlama: {0}", yukari_yuvarlama);
        Console.WriteLine("Aşağı yuvarlama: {0}", asagi_yuvarlama);
        Console.WriteLine("Yakın yuvarlama: {0}", yakin_yuvarlama);

Güç alma:


        double taban = 2;
        double us = 3;

        double sonuc = Math.Pow(taban, us);
        Console.WriteLine("Sonuç: {0}", sonuc); // Çıktı: 8

Logaritma:


        double sayi = 10;
        double taban = 2;

        double logaritma = Math.Log(sayi, taban);
        Console.WriteLine("Logaritma: {0}", logaritma); // Çıktı: 3.302775

Trigonometrik fonksiyonlar:


        double aci = Math.PI / 4; // 45 derece

        double sinüs = Math.Sin(aci);
        Console.WriteLine("Sinüs: {0}", sinüs); // Çıktı: 0.70710678

        double kosinüs = Math.Cos(aci);
        Console.WriteLine("Kosinüs: {0}", kosinüs); // Çıktı: 0.70710678

        double tanjant = Math.Tan(aci);
        Console.WriteLine("Tanjant: {0}", tanjant); // Çıktı: 1

Mutlak değer:


        double sayi = -10;

        double mutlak_deger = Math.Abs(sayi);
        Console.WriteLine("Mutlak değer: {0}", mutlak_deger); // Çıktı: 10

Karekök:

        double sayi = 9;

        double karekok = Math.Sqrt(sayi);
        Console.WriteLine("Karekök: {0}", karekok); // Çıktı: 3
