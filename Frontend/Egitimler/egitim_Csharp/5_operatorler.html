Operatörler

Operatörler, programda veri işlemleri gerçekleştirmek için kullanılan özel sembollerdir. Farklı türde operatörler, farklı işlemler gerçekleştirmek için kullanılır. C# dilinde birçok farklı operatör türü bulunur.

Temel Operatörler:

  • Aritmetik Operatörler: Toplama (+), çıkarma (-), çarpma (*), bölme (/), mod alma (%), üs alma (^)
  • Mantıksal Operatörler: Ve (&&), veya (||), değil (!), xor(^)
  • Karşılaştırma Operatörleri: Eşittir (==), eşit değildir (!=), büyükse (>), büyükse veya eşitse (>=), küçüktür (<), küçüktür veya eşitse (<=)
  • Atıf Operatörleri: Atama (=), artırma (++), azaltma (--), toplama ataması (+=), çıkarma ataması (-=), çarpma ataması (*=), bölme ataması (/=), mod alma ataması (%=)
  • Diğer Operatörler: Koşullu operatör (?:), boş operatör (null), tip döküm operatörü (as, is)

Örnek Kullanımlar


        /*Rakamsal Atamalarda*/
        int x = 10; // 'x' değişkenine 10 değeri atanır
        int y = 5;  // 'y' değişkenine 5 değeri atanır

        /*Matematiksel İşlemlerde*/
        int toplam = x + y; // Toplama işlemi
        int fark = x - y;   // Çıkarma işlemi
        int carpim = x * y; // Çarpma işlemi
        int bolum = x / y;  // Bölme işlemi
        int mod = x % y;    // Mod alma işlemi
        int us = x ^ y;     // Üs alma işlemi

        /*Karşılaştırma İşlemlerinde*/
        bool esitMi = x == y; // Eşitlik karşılaştırma
        bool esitDegilMi = x != y; // Eşitsizlik karşılaştırma
        bool buyukMu = x > y;  // Büyüklük karşılaştırma
        bool buyukEsitMi = x >= y; // Büyük veya eşit karşılaştırma
        bool kucukMu = x < y;  // Küçüklük karşılaştırma
        bool kucukEsitMi = x <= y; // Küçük veya eşit karşılaştırma

        /*Mantıksal İşlemlerde*/
        bool veIslemi = x > 5 && y < 10; // Ve (&&) mantıksal operatörü
        bool veyaIslemi = x == 10 || y == 5; // Veya (||) mantıksal operatörü
        bool degilIslemi = !(x == y);  // Değil (!) mantıksal operatörü
        bool xorIslemi = x > 5 ^ y < 10; // Xor(^) mantıksal operatörü

        /*Matematiksel İşlem Atamalarında*/
        x += 2;  // Toplama ataması
        x -= 1;  // Çıkarma ataması
        x *= 3;  // Çarpma ataması
        x /= 2;  // Bölme ataması
        x %= 4;  // Mod alma ataması

        /*Metinsel Atamalarda*/
        string isim = "Merhaba"; // "Merhaba" metni 'isim' değişkenine atanır
        int uzunluk = isim.Length; // 'isim' değişkeninin uzunluğu 'uzunluk' değişkenine atanır
        
        /*Koşullu İfadeler İçinde Kullanım*/
        if (x > 0) // 'x' 0'dan büyükse
        {
            Console.WriteLine("x değeri 0'dan büyüktür.");
        }
        else // 'x' 0'dan küçük veya eşitse
        {
            Console.WriteLine("x değeri 0'dan küçük veya eşittir.");
        }

        string sonuc = (x > y) ? "x, y'den büyüktür" : "x, y'den küçük veya eşittir"; // Koşullu operatör
