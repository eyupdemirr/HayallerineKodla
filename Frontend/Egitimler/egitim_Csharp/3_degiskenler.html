Değişkenler

Değişkenler, programda değerleri depolamak için kullanılan adlandırılmış hafıza alanlarıdır. Değişkenler sayesinde program kodumuzda tekrarlayan verileri dinamik olarak yönetebilir ve programın işleyişini daha esnek hale getirebiliriz.

Değişken Türleri

C# dilinde, değişkenleri tanımlamak için farklı veri türleri kullanılabilir. En yaygın kullanılan veri türleri şunlardır:

  • int: Tam sayılar için kullanılır.
  • double: Ondalık sayılar için kullanılır.
  • string: Metin (karakter dizileri) için kullanılır.
  • bool: Doğru veya yanlış değerleri (true veya false) için kullanılır.
  • char: Tek bir karakter için kullanılır.

Örnekler:


        int x = 10;                // Tam sayı değişkeni
        double y = 3.14;      // Ondalık sayı değişkeni
        string isim = "Ali";    // Metin değişkeni
        bool aktif = true;      // Doğru/yanlış değişkeni
        char karakter = 'A';  // Tek karakter değişkeni

Değişken İşlemleri

Değişken Tanımlama

Değişken tanımlamak için aşağıdaki genel sözdizimi kullanılır:


        veri_tipi değişken_adı = değer;
  • veri_tipi: Değişkenin depolanacağı veri türünü belirtir.
  • değişken_adı: Değişkene verilecek ad.
  • değer: Değişkene ilk atanan değer (isteğe bağlı).

Sabit Değişkenler

Sabit değişkenler, program boyunca değeri değişmeyen özel bir değişken türüdür. Değerlerini program çalışırken değiştirilemezler. Sabit değişkenler, program kodunda tutarlılık ve güvenlik sağlamak için kullanılır.


        const veri_tipi sabit_isim = değer;
  • Sabit değişkenlerin değerleri program çalışırken değiştirilemez.
  • Sabit değişkenler, büyük/küçük harf duyarlıdır.
  • Sabit değişkenler, tanımlandıkları yerde değerlendirilmelidir.

Değişken Adı Kuralları:

  • Değişken adları büyük/küçük harf duyarlıdır.
  • Değişken adları sayı ile başlayamaz.
  • Değişken adları boşluk içeremez.
  • Değişken adları özel karakterler (örneğin: !, @, #) içeremez.

Değişken Kullanımı

Değişkenler, program kodumuzda çeşitli amaçlar için kullanılabilir:

  • Değerleri depolamak: Bir değeri bir değişkene atayarak, programın ilerleyen bölümlerinde o değeri kullanabiliriz.
  • Hesaplamalar yapmak: Değişkenler, aritmetik ve mantıksal işlemlerde kullanılabilir.
  • Verileri kullanıcıdan almak: Kullanıcıdan veri girişi için değişkenler kullanılabilir.
  • Verileri bir dosyaya yazmak: Değişkenlerin değerleri, bir dosyaya yazılabilir.

Değişken Kapsamı

  • Yerel değişkenler: Bir fonksiyon veya blok içerisinde tanımlanan değişkenlerdir. Sadece o fonksiyon veya bloğun içinde erişilebilirler. 
    • 
        void Islem()
        {
          int x = 10; // Yerel değişken
        
          Console.WriteLine("x: " + x); // Yerel değişkene erişim
        }
        
        void Main()
        {
          // x değişkenine erişilemez (yerel kapsam)
        }
  • Küresel değişkenler: Tüm program boyunca erişilebilen değişkenlerdir. Herhangi bir fonksiyon veya bloğun dışında tanımlanırlar. 
    • 
        int x = 20; // Küresel değişken

        void Islem()
        {
          Console.WriteLine("x: " + x); // Küresel değişkene erişim
        }

        void Main()
        {
          Console.WriteLine("x: " + x); // Küresel değişkene erişim
        }
